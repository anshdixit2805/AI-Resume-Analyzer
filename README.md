# AI Resume Analyzer

AI Resume Analyzer is a web-based application that evaluates resumes and provides insights such as skill matching, resume score, and job compatibility.

## Features
- Resume parsing using NLP
- Skill extraction
- Job matching score
- Resume feedback and improvement suggestions

## Tech Stack
- Python
- Flask
- HTML/CSS
- Machine Learning / NLP

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open in browser:
   http://localhost:5000
